# Football League (RPL) SQL Analytics

Проект по проектированию реляционной базы данных и аналитическому SQL-анализу данных футбольной лиги РПЛ в PostgreSQL.

## Цель проекта

Показать навыки проектирования БД и аналитической работы с SQL:
- создание нормализованной структуры данных
- работа с внешними ключами и ограничениями целостности
- написание аналитических запросов с JOIN, CTE, агрегатами и оконными функциями

## Структура базы данных

В проекте используются следующие сущности:
- `teams` — команды
- `stadiums` — стадионы
- `players` — игроки
- `coaches` — тренеры
- `seasons` — сезоны
- `matches` — матчи
- `goals` — голы
- `match_events` — игровые события

## Что реализовано

### Проектирование БД
- связи 1:1, 1:N и рекурсивная связь
- первичные и внешние ключи
- ограничения целостности
- нормализованная структура таблиц

### Аналитические запросы
- топ бомбардиров и ассистентов
- результативность команд
- домашняя и гостевая статистика
- анализ зарплат игроков и команд
- ранжирование игроков по зарплате внутри команды
- возрастной профиль составов
- дисциплинарная статистика
- связь бюджета и результативности

## Используемые технологии

- PostgreSQL
- SQL
- CTE
- Window Functions
- Aggregations
- CASE / Subqueries / JOIN

## Как запустить

1. Создать пустую базу данных в PostgreSQL
2. Выполнить `schema.sql`
3. Выполнить `seed_clean.sql`
4. Выполнить `analysis.sql`
